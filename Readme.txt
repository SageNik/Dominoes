                                            ***  D O M I N O E S  ***

     Данная программа предназначена для создания наборов домино из случайно выбранного (или заданного) количества костей
 домино из общего пула (стандартный набор для игры в домино 28 костей) и выстраивания всех возможных цепочек (или только
 самой длинной из них) из созданного набора. Данные о наборах и созданных из них цепочех сохраняются в базу данных.

                                             * Запуск программы *

     Для запуска программы необходимо иметь установленные Java 1.7+ Maven (тестировалось на версии 3.5.2)

     В консоле: (находясь в папке проекта) выполнить комманды:
                mvn clean install
                mvn jetty:run-war

     В среде разработки: в файле pom.xml закомментировать <scope>provided</scope> в зависимостях с artifactId:
        jetty-webapp, jetty-jsp  и запускать через main class ua.domino.launcher.Launcher


        При компиляции программы база данных создаётся автоматически.

        В браузере приложение открывается по адресу: http://localhost:8080/
